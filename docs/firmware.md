# 13. Flashing Firmware

The Challenger runs [ZMK](https://zmk.dev). Flash the firmware before powering on for the first time.

1. Plug the nice!nano into your computer via USB-C.
2. Double-tap the reset button quickly. The nice!nano will appear as a USB drive called `NICENANO`.
3. Drag and drop the `.uf2` firmware file onto the `NICENANO` drive. Your OS may report an error when the transfer finishes; this is normal and can be ignored.
4. The drive will disappear and the nice!nano will reboot automatically.
5. Slide the power switch to **ON**.

> **Note:** If the `NICENANO` drive doesn't appear after double-tapping reset, try tapping faster. If the nice!nano has never been flashed before, a single tap is enough.
