# 3. Reset Button

![Reset button placed on the PCB](../assets/03-reset-button-placed.jpg)
![Reset button soldered](../assets/03-reset-button-soldered.jpg)

The reset button (Panasonic EVQ-PUL02K) goes on the **top side** of the PCB.

1. Place the reset button in its footprint. The legs only fit in one orientation.
2. Solder all four legs.

> **Verify:** Press the button. It should click and spring back.
