# 2. Hotswap Sockets

![Hotswap sockets placed on the PCB](../assets/02-hotswap-placed.jpg)
![Hotswap sockets soldered](../assets/02-hotswap-soldered.jpg)

The 35 Kailh MX hotswap sockets go on the **back side** of the PCB, in the outlined footprints. The socket shape matches the footprint outline, so orientation is clear.

1. Place all sockets into their footprints.
2. Solder one pad on each socket first, pressing the socket flat with a finger or tweezers as you go. Sockets can pop up from the heat if not held down.
3. Go back and solder the second pad on all sockets.

> **Verify:** Press each socket with a fingernail. It should not move or wiggle. Check that both pads have solid, shiny joints with no bridges between them.
