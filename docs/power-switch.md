# 4. Power Switch

![Power switch placed on the PCB](../assets/04-power-switch-placed.jpg)
![Power switch soldered](../assets/04-power-switch-soldered.jpg)

The slide switch (Alps Alpine SSSS811101) goes on the **top side** of the PCB.

1. Place the switch in its footprint. The legs are asymmetric so it only fits one way.
2. Solder all the legs.

> **Verify:** Slide the switch back and forth. It should move smoothly between both positions.
